## Excel Formatting Standards (MANDATORY)
- **Font:** Montserrat, size 12 for ALL body and header content
- **Titles:** Montserrat Bold, size 18, color Rose Dark (#A66B72)
- **Subtitles:** Montserrat Bold, size 14, color Gold (#C8A96E)
- **Headers:** Montserrat Bold, size 12, white text on Rose Dark background
- **Body text:** Montserrat Regular, size 12, color Text (#2D2420)
- **Alignment:** ALL cells center-center (horizontal center, vertical center)
- **Wrap text:** ALWAYS enabled on every cell
- **Row heights:** Minimum 35px for headers, 50px for body with multiline content
- **Borders:** Thin borders, color #E8DED4
- **Filters:** Auto-filters on ALL header rows for navigation
- **Freeze panes:** Always freeze header rows
- **Tab colors:** Use brand colors (Rose Dark, Gold, Green) for sheet tabs
- **Ronda rows:** Highlight with FFF5E6 background + Rose Dark bold font

## Presentation Standards
- Use brand gradient for title slides: linear-gradient(135deg, #A66B72, #C4878E, #DDB3B7)
- Gold (#C8A96E) for accent elements and secondary headings
- Montserrat font throughout
- Clean, minimal layouts with generous whitespace

## Document Standards
- Montserrat font, size 12 for body
- Rose Dark (#A66B72) for headings
- Gold (#C8A96E) for subheadings
- Include 🌿 emoji with Ronda branding where appropriate
